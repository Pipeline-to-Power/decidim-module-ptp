# Decidim::Sms-twilio

Twilio SMS integration.

## Usage

Sms-twilio will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-sms-twilio"
```

And then execute:

```bash
bundle
```
